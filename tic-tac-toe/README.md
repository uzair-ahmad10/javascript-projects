# Tic Tac Toe

A classic Tic‑Tac‑Toe game built with vanilla JavaScript, HTML, and CSS.

## Features
- Two‑player mode (local).
- Highlight winning line.
- Reset button to start a new game.
- Responsive layout for desktop and mobile.

## How to Play
1. Open `index.html` in a browser.
2. Click on an empty cell to place your mark (X or O).
3. Alternate turns until a player wins or the board fills (draw).
4. Press **Reset** to play again.

## Customisation
Edit `script.js` to change the AI logic, board size, or add a single‑player mode.
