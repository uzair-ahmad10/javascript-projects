# Drag and Drop

A vanilla JavaScript drag‑and‑drop demo that lets you move items between containers.

## Features
- Drag items with mouse or touch.
- Drop targets highlight on hover.
- Simple, dependency‑free implementation.

## How to Run
1. Open `index.html` in a browser.
2. Drag the colored boxes into the target area.

## Customisation
Edit `script.js` to change the draggable items or add new drop zones.
