# Quiz App

A simple, interactive **Quiz Application** built with vanilla JavaScript, HTML, and CSS. Users can answer multiple-choice questions, receive immediate feedback, and see their final score.

## Features
- Multiple-choice questions
- Immediate feedback after each answer
- Score tracking and final results display
- Responsive design using CSS

## Demo
Open `index.html` in a web browser to start the quiz.

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. The quiz will load and you can start answering questions.
