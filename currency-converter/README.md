# Currency Converter

A simple JavaScript web app that converts amounts between different currencies using real‑time exchange rates.

## Features
- Fetches live rates from an API.
- Supports multiple currencies.
- Responsive UI built with vanilla HTML/CSS/JS.

## Setup
1. Open `index.html` in a browser.
2. No build step required; all logic runs client‑side.

## Usage
Select the source and target currencies, enter an amount, and click **Convert**. The converted amount will be displayed instantly.
