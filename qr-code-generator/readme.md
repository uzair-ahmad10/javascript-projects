# QR Code Generator

A simple web-based QR code generator built with vanilla JavaScript, HTML, and CSS. Users can enter a URL or plain text, click **Generate**, and instantly see a QR code that can be downloaded.

---

## ✨ Features
- Real‑time QR code generation
- Download QR code as PNG
- Responsive design with a modern UI
- No external libraries required (uses the free QRCode API)

## 📦 Installation & Development
1. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd qr-code-generator
   ```
2. **Open `index.html`** in any modern browser (no server required).
3. **Edit the source** – Files are located at:
   - `index.html` – markup and layout
   - `style.css` – styling (glassmorphism, dark mode, micro‑animations)
   - `script.js` – QR generation logic

---

## 🛠️ Usage
1. Type a URL or any text into the input field.
2. Click **Generate**.
3. The QR code appears below the button.
4. Click **Download** to save the QR code as a PNG image.
