# JavaScript Projects Collection

Welcome to the **JavaScript Projects** repository! This folder contains a collection of small, vanilla‑JavaScript web applications that demonstrate core web‑development concepts without any external libraries or frameworks.

## Projects Overview

| Project | Description | Link |
|---------|-------------|------|
| **Currency Converter** | Convert amounts between currencies using live exchange‑rate data. | [README](./currency-converter/README.md) |
| **Drag and Drop** | Interactive drag‑and‑drop demo with customizable items and drop zones. | [README](./drag-drop/README.md) |
| **Tic Tac Toe** | Classic two‑player Tic‑Tac‑Toe game with win detection and reset functionality. | [README](./tic-tac-toe/README.md) |
| **Digital Clock** | Digital Clock with Gradient color UI. | [README](./digital-clock/README.md) |

## How to Explore
1. Open any project's folder.
2. Open the `README.md` inside that folder for detailed usage instructions.
3. Open the `index.html` (or equivalent) in a web browser to run the demo.
